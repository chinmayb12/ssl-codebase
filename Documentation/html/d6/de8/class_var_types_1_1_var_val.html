virtual

Creates a clone of the item. Note, that for containers that include VarType pointers (e.g. a list consisting of child-elements), this will perform a SHALLOW-COPY (e.g. the pointer values will be copied, but not the actual children references by them.), To perform a fully recursive deep-copy clone, run "deepClone()".
