Detailed Description

PassReceive accepts a receive_point as a parameter and gets setup there to catch the ball It transitions to the 'aligned' state once it's there within its error thresholds and is steady Set its 'ball_kicked' property to True to tell it to dynamically update its position based on where the ball is moving and attempt to catch it.

It will move to the 'completed' state if it catches the ball, otherwise it will go to 'failed'.
