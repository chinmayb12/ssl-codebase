Detailed Description

template<class CLASS_VARVAL_TYPE, VarTypeId TPL_vartype_id>
class VarTypes::VarProtoBufferVal< CLASS_VARVAL_TYPE, TPL_vartype_id >

A Vartype that wraps around google protocol buffers.

Author
Stefan Zickler, (C) 2008
See also
VarTypes.h

The VarProtoBufferVal Vartype allows wrapping of Google Protocol Buffer Messages. It will automatically ensure that the data will be converted to/from base64 when storing/loading to/from XML.

If you don't know what VarTypes are, please see VarTypes.h
