Detailed Description

A Vartype for storing binary data.

Author
Stefan Zickler, (C) 2008
See also
VarTypes.h

The VarBlob Vartype allows for storage of any continuous binary string. It will automatically ensure that the data will be converted to/from base64 when storing/loading to/from XML.

Note, that to use this VarType, you will first have to manually allocate a data-block and supply that data-pointer and a max-size as arguments to the VarBlob.

Alternatively, you can use its allocate(size) function to construct the data-block

If you don't know what VarTypes are, please see VarTypes.h
