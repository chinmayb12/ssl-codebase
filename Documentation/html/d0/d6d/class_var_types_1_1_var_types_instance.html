Detailed Description

A singleton wrapper to VarTypesFactory.

Author
Stefan Zickler, (C) 2008
See also
VarTypes.h

This is a singleton wrapper to the VarTypesFactory This allows global registration of a custom VarTypesFactory subclass, by calling VarTypesInstance::setFactory(customVarTypesFactorySubClass) before using any other vartypes functions. This in turn provides the simple ability to add additional types to the vartypes system without touching the core files. If setFactory is not called before getFactory() is used, then the VarTypes' default factory class "VarTypesFactory" will be auto-instanciated instead.

If you don't know what VarTypes are, please see VarTypes.h
