Detailed Description

The base class of the VarTypes system.

Author
Stefan Zickler, (C) 2008
See also
VarTypes.h

This is is the base-class of the VarType system. All other VarTypes must publicly inherit this class.

If you don't know what VarTypes are, please see VarTypes.h
