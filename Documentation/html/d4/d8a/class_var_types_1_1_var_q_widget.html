Detailed Description

An Vartype for embedding QWidgets into the Var-tree.

Author
Stefan Zickler, (C) 2008
See also
VarQWidget
VarTypes.h

Note, that this class breaks the model/view paradigm as it will construct a single qwidget which will be shared between viewers and not separate instances for each viewer. This is fine if you only have a single viewer, but will likely create problems if you have multiple ones.

To avoid these issues, it is better to simply inherit VarType and create a qwidget factory by reimplementing VarType's createEditor(...);

If you don't know what VarTypes are, please see VarTypes.h
