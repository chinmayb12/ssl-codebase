Detailed Description

XML helper functions of the VarType system.

Author
Stefan Zickler, (C) 2008
See also
VarTypes.h

This class provides static functions to parse or write an XML file representing a VarTypes tree.

If you don't know what VarTypes are, please see VarTypes.h

read a VarType-tree from an xml file and return its root nodes as a vector.

Parameters
existing_nodesrepresents any pre-defined vartypes tree and should be an empty vector if no tree pre-exists.

If a tree does pre-exist then the read function will update the data of any existing nodes, or create new nodes if they are missing in the existing tree.
