Detailed Description

The 'view' using a QTreeView for displaying VarTypes in the QT4 Item-Model.

Author
Stefan Zickler, (C) 2008
See also
VarTypes.h

This file is part of the QT4 Item-Model for VarTypes. It is used when you want to edit/visualize VarTypes using QT4's model/view system.

If you don't know what VarTypes are, please see VarTypes.h
