Detailed Description

A v4l-based USB/Video For Linux Capture Class.

Author
Eric Zavesky, (C) 2016

This class mirrors popular v4l implementations giving the most compatible options and based on the DC1394 base classes by Stefan.

Overall, it not only provides capture-abilities, but also full on-the-fly configuration abilities through the VarTypes system.

If you find your camera not working correctly, or discover a bug, please inform the author, but also consider contributing yourself, so that we can cover as many options as possible.
