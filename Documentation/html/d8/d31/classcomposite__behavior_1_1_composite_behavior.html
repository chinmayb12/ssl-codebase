def composite_behavior.CompositeBehavior.spin_cb (   self)

Override StateMachine.spin() so we can call spin() on subbehaviors.

Spin Composite Behavior,execute all behaviors inside composite behavior

All behavior inside Composite behavior instance is executed using threading. If subbehavior inside Composite behavior is Composite subbehavior, then spin_cb() is used else spin().

See also
Behavior::spin()
