Detailed Description

This is a Trigger-like VarType of the VarTypes system.

Author
Stefan Zickler, (C) 2008
See also
VarTypes.h

This vartype will be represented as a single button in the GUI. If clicked, the trigger() slot will be called and an internal counter will be incremented by one.

If you don't know what VarTypes are, please see VarTypes.h
